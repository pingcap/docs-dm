# DM 告警信息

使用 TiUP 部署 DM 集群的时候，会默认部署一套[告警系统](migrate-data-using-dm.md#第-8-步监控任务与查看日志)。

DM 的告警规则及其对应的处理方法可参考[告警处理](dm-handle-alerts.md)。

DM 的告警信息与监控指标均基于 Prometheus，告警规则与监控指标的对应关系可参考 [DM 监控指标](monitor-a-dm-cluster.md)。
